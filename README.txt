===== Flavor Finder =====
Flavor Finder adalah program search engine sederhana yang dikembangkan untuk mencari nama resep makanan dari seluruh dunia yangmenggunakan Algoritma Fuzzy dalam proses pencarian untuk memberikan hasil yang relevan dan toleran terhadap kesalahan pengejaan atau variasi nama resep. API yang digunakan pada program adalah API yang disediakan oleh TheMealDB.

Cara menjalankan program:
1. Buka file melalui code editor
2. Buka terminal dan ketik command: python app.py
3. Cari dan klik url berikut pada terminal: http://127.0.0.1:5000
4. Program akan berjalan pada browser dan siap untuk digunakan
